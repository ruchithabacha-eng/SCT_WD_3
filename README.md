# Quiz Application

An interactive and user-friendly Quiz Application built using HTML, CSS, and JavaScript. Users can answer multiple-choice questions and receive their final score at the end of the quiz.

## 🚀 Features

- Multiple-choice questions
- One question displayed at a time
- Instant option selection
- Final score displayed after completing the quiz
- Responsive and clean user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
Quiz-Application/
│── index.html
│── style.css
│── script.js
│── README.md
```

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your web browser.
